# GŁĘBIA — scroll-driven 3D demo

Jednoplikowa strona 3D sterowana scrollem: zejście z powierzchni oceanu na dno Rowu Mariańskiego.

- Three.js (WebGL) — kamera, mgła i światło sterowane pozycją scrolla
- Śnieg morski (particles), bioluminescencja, wireframe'owe stworzenia, dno z szumem
- HUD komputera nurkowego (głębokość / ciśnienie / temperatura) liczony ze scrolla
- `prefers-reduced-motion` respektowane

Napisane w całości przez Claude (Anthropic) w jednej rozmowie — jako dowód, że scroll-driven strony 3D z rolek na IG to realna, powtarzalna rzecz, a nie montaż.

**Live:** https://ogarniamyai.github.io/glebia/
