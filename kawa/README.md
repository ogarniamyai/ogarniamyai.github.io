# PALARNIA·9 — coffee scroll-driven demo

Odpowiedź na viralowe rolki „AI zrobiło stronę kawy": w pełni działająca, proceduralna strona 3D bez ani jednego zdjęcia czy modelu z internetu.

- szklany kubek + kawa (lathe, MeshPhysicalMaterial), powierzchnia z wirami cremy i rozetą (shader)
- para z kubka (shader particles), proceduralne drewno blatu (shader), bokeh świateł kawiarni
- **scroll = parzenie**: ziarna zlatują się, mleko wlewa się dokładnie tyle, ile przewiniesz (kolor kawy jaśnieje), kostka cukru spada i rozbryzguje się — cofniesz scroll, krople wracają
- HUD: temperatura, ml mleka, stan napoju
- Three.js r128, jeden plik, napisane w całości przez Claude

**Live:** https://ogarniamyai.github.io/kawa/
